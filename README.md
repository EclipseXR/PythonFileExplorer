# PythonFileExplorer 

**A File Explorer/Finder made in Python. Can be used to store files.**

*100% Complete*

### Details
- Contains Five Folders
  - Pictures, Music, Documents, Trash, and Desktop
  - Simple File Viewer
  - Can create files as well as open them in their default programs
